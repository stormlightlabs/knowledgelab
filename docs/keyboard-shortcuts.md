# Keyboard Shortcuts

## File Operations

- `Cmd/Ctrl+N`: Create new note
- `Cmd/Ctrl+S`: Save (auto-saves by default)

## Navigation

- `Cmd/Ctrl+K`: Command palette / search
- `Cmd/Ctrl+Shift+L`: Note list view
- `Cmd/Ctrl+Shift+G`: Graph view

## Formatting (in editor)

- `Cmd/Ctrl+B`: Bold text
- `Cmd/Ctrl+I`: Italic text
- `Cmd/Ctrl+E`: Inline code
- `Cmd/Ctrl+K`: Insert/edit link
- `Cmd/Ctrl+1-6`: Set heading level

## Panels

- `Cmd/Ctrl+B`: Toggle backlinks
- `Cmd/Ctrl+Shift+F`: Toggle search panel
- `Cmd/Ctrl+Shift+T`: Toggle tags panel
- `Cmd/Ctrl+Shift+P`: Toggle preview mode
